# Grocery Store Management System (SQL)

A SQL-based database project that simulates the operations of a grocery store. The system manages products, customers, suppliers, employees, and orders while enabling data analysis through SQL queries to generate meaningful business insights.

## Project Overview

This project demonstrates the design and implementation of a relational database using SQL. It includes normalized tables, relationships, constraints, and analytical queries to manage store operations and analyze sales performance.

## Features

- Manage products, categories, suppliers, customers, employees, and orders
- Relational database with Primary Keys and Foreign Keys
- One-to-Many relationships between tables
- Sales and revenue analysis
- Customer purchasing behavior analysis
- Employee performance tracking
- Supplier contribution analysis
- Product performance insights

## Technologies Used

- SQL (MySQL)
- MySQL Workbench
- ER Diagram
- CSV Dataset

1. Open MySQL Workbench.
2. Execute the GROCERY STORE.sql script.
